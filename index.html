Wake-on-LAN Service

Devices

Wake-up directly using HTTP Requests

/wolweb/wake/<Device Name> - Returns a JSON object
{
    "success":true,
    "message":"Sent magic packet to device Server with Mac 34:E6:D7:33:12:71 on Broadcast IP 192.168.1.255:9",
    "error":null
}
Parameter Value
Method

GET
Success   Type: BOOLEAN  Returns a value based on whether magic packet was sent
Message   Type: STRING Returns a description of the action taken
Error   Type: JSONObject Returns a errors as a JSON converted from any raised GOLANG Error Objects
